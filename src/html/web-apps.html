React Router

"Yootoob" is a React application that utilizes a Youtube Rapid API to fetch data and it displays the information on the DOM. The CSS has been written with Material UI and the application is dynamic for mobile view. Yootoob allows the user to brows video categories, view videos, search for videos, and view channel pages. Each video displays not only the video itself but also the video description as well as the view and like count.
